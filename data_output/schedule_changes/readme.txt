Columns:
Route - route name
Old - number of trips for route based on GTFS schedules from 2020/03/03, sampled across week 2020/03/09-2020/03/16
New - number of trips for route based on GTFS schedules from 2022/12/28, across across week 2023/01/09-2023/01/09
changePct - percent of service added (positive) or removed (negative) in New column


Files:
separated by Bus/L.
Sampled for a Wednesday (weekday service, theoretically each workday gets its own schedule but the differences between them are minimal), Saturday and Sunday.
TOTAL_WEEKLY file has all trips from Monday to Sunday sumed up for an overall comparison
